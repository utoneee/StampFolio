# StampFolio

集郵冊挑戰打卡 app —— 序列制抽卡＋蓋章打卡，換皮（theme pack）架構讓同一引擎可以出多本不同主題的集郵冊。

## 命名

- **StampFolio**：專案代號／系列名。
- **StampFolio: Daily**（日常×文青）：第一本，日常挑戰打卡。
- **StampFolio: Studio** 或 **StampFolio: Pixel**（創作×8bit）：第二本，創作練習企劃（見 `05-下一本-創作練習企劃.md`），命名待定。

系列以「StampFolio: {子標}」延伸，換皮＝換 theme pack（素材資料夾＋挑戰庫 JSON＋主題設定），不改程式。
